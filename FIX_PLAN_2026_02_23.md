### P0-2: ai-access-control.ts — Rate Limits Completely Bypassed

**File**: `server/middleware/ai-access-control.ts:616-623`
**Category**: Security / Stub
**Root Cause**: `getRecentFeatureUsage()` always returns `[]`, so burst limit checks (`recentBurst.length >= feature.rateLimit.burstLimit`) never trigger.

**Current Code** (lines 616-623):
```typescript
private static async getRecentFeatureUsage(userId, featureId, since): Promise<Array<{timestamp: number}>> {
  // Mock implementation - in production would query actual usage data
  return [];
}
```

**Problem**: ALL per-feature rate limits are disabled. Any user can make unlimited AI calls.

**Fix**: Implement in-memory sliding window rate counter (no schema change needed):
```typescript
// Add at class level:
private static rateLimitStore = new Map<string, number[]>();

private static async getRecentFeatureUsage(
  userId: string, featureId: string, since: number
): Promise<Array<{ timestamp: number }>> {
  const key = `${userId}:${featureId}`;
  const timestamps = this.rateLimitStore.get(key) || [];
  // Prune expired entries
  const valid = timestamps.filter(t => t >= since);
  this.rateLimitStore.set(key, valid);
  return valid.map(t => ({ timestamp: t }));
}

// Call this after each AI request succeeds (in the middleware's proceed path):
private static recordUsage(userId: string, featureId: string): void {
  const key = `${userId}:${featureId}`;
  const timestamps = this.rateLimitStore.get(key) || [];
  timestamps.push(Date.now());
  this.rateLimitStore.set(key, timestamps);
}
```

**Architecture Fit**: In-memory is acceptable for dev/single-instance. For production with multiple instances, migrate to Redis (already available via ioredis when `REDIS_ENABLED=true`). This can be done later as a P2 enhancement.

**Test Criteria**:
- Make N+1 requests where N = burst limit
- Verify the (N+1)th request is rejected with 429
- Verify rate counter resets after window expires

private static async getCacheValue(key: string): Promise<any> {
  if (process.env.REDIS_ENABLED === 'true') {
    const redis = (await import('../redis-client')).default;
    const val = await redis.get(`ai-cache:${key}`);
    return val ? JSON.parse(val) : null;
  }
  const entry = this.cacheStore.get(key);
  if (!entry || Date.now() > entry.expiry) {
    this.cacheStore.delete(key);
    return null;
  }
  return entry.value;
}

private static async setCacheValue(key: string, value: any, ttl: number): Promise<void> {
  if (process.env.REDIS_ENABLED === 'true') {
    const redis = (await import('../redis-client')).default;
    await redis.setex(`ai-cache:${key}`, ttl, JSON.stringify(value));
    return;
  }
  this.cacheStore.set(key, { value, expiry: Date.now() + ttl * 1000 });
}
